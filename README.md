# پورتال کستینگ (HTML + وردپرس)

ثبت‌نام با سه نقش (هنرجو / کارگردان / تهیه‌کننده) و دو درگاه ورود (هنرجو / کارفرما).
کاربران در همان دیتابیس وردپرس ذخیره می‌شوند.

## نصب روی هاست

1. کل پوشه `casting-portal` را داخل ریشه وردپرس آپلود کنید (کنار `wp-config.php` و `wp-load.php`).
2. اگر مسیر فرق دارد، فایل `config.php` را باز کنید و `CASTING_WP_LOAD` را اصلاح کنید.
3. آدرس را باز کنید، مثلاً:
   `https://yourdomain.com/casting-portal/`
4. از منوی وردپرس به همین آدرس لینک بدهید.

## صفحات

| فایل | کار |
|------|-----|
| `index.php` | صفحه اصلی + دو درگاه ورود |
| `register.php` | ثبت‌نام + انتخاب نقش |
| `login-talent.php` | ورود هنرجو |
| `login-employer.php` | ورود کارفرما |
| `dashboard-talent.php` | پنل هنرجو |
| `dashboard-employer.php` | پنل کارفرما |
| `profile-talent.php` | پروفایل + عکس + لینک ویدیو |
| `talents.php` | لیست و فیلتر هنرجویان (کارفرما) |
| `talent.php` | مشاهده پروفایل یک هنرجو |

## نقش‌ها

- `talent` → فقط از ورود هنرجو
- `director` / `producer` → فقط از ورود کارفرما

نقش در `user_meta` با کلید `casting_role` ذخیره می‌شود.

## تغییر نام برند

در `config.php` مقدار `CASTING_BRAND` را عوض کنید.
